Tantangan Implementasi Numerics
Coba asah kemampuan Anda, carilah data numerics dengan nilai distance_from_tokyo = 293.6.

SELECT *
FROM shinkansen_station
WHERE distance_from_tokyo = 293.6;

Tantangan Implementasi Non-numeric
Coba asah kemampuan Anda dengan mencari data non-numeric dari kolom station_name Atami seperti berikut.

SELECT *
FROM shinkansen_station
WHERE station_name = 'Atami';
